# Audio Visualizer
![LOGO](https://images.weserv.nl/?url=https://article.biliimg.com/bfs/new_dyn/13772b609241784997fd93b5daec38496741540.png "Audio Visualizer")

A React-based audio visualization tool.

![Screenshot](https://images.weserv.nl/?url=https://article.biliimg.com/bfs/new_dyn/25e19f4894ee6759c2ad5c54d289ea056741540.png "Audio Visualizer")

## Features

- Upload audio files and visualize them in real time
- Multiple visualization types (radial, circle, waveform, etc.)
- Export visualizations as video
- Responsive and modern UI

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Usage

1. Upload your audio file.
2. Choose a visualization type.
3. Play and preview the animation.
4. Export the visualization as a video.
