# Audio Visualizer

A React-based audio visualization tool.

## Features

- Upload audio files and visualize them in real time
- Multiple visualization types (radial, circle, waveform, etc.)
- Export visualizations as video
- Responsive and modern UI

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Usage

1. Upload your audio file.
2. Choose a visualization type.
3. Play and preview the animation.
4. Export the visualization as a video.
